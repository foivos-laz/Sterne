The app if synced in Android Studio for compilation will not compile due to critical files not being present for security reasons. The app does work and compile from our end, with those files present. You may download it in releases, where the app is functional if installed on an Android device!

HERA uses Firabase Authentication, Mapbox, as well as Firebase Firestore, since during developement we didnt have access to HUAWEI credits, which we received the day before the European final. Therefore it was not possible to migrate everything in a day to another cloud solution.

Features as of righ now:
1. Panic Button
2. AI Speak (Not funtional, requires access to Huawei Cloud)
3. Dangerous Areas
4. Community Posts
5. Nearest Open Store/ Safe Havens (Not funtional, requires Huawei Map API access and a new platform for registered stores)
